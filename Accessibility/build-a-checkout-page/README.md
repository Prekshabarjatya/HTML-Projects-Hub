# Accessible Checkout Page

A simple, accessible HTML checkout page for an online store.

## Overview

The page displays a shopping cart section with a product image, description, and price, followed by a semantically structured payment form. Accessibility features support screen readers and keyboard navigation, using labels and descriptive helper text for guidance.

## Features

- **Semantic HTML**: Structured with headings, sections, figure, and form tags for clarity and accessibility.
- **Accessible Cart Display**: Product image includes descriptive alt text for visually impaired users.
- **Payment Form**: Input fields are clearly labeled, marked required, and include instructions via `aria-describedby` for card number entry
- **Helper Text**: Visible instructions guide proper card number entry.
